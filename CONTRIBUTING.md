# Contributing to Free PDF Tools by Toolimi

Thank you for your interest in contributing! We welcome pull requests, bug reports, and feature suggestions.

## How to Contribute

- **Suggest a new PDF tool:** Open an issue and describe the tool you'd like us to add.
- **Fix or improve this README:** Fork this repo, make your changes, and submit a pull request.
- **Report a bug:** Use the Issues tab with clear steps to reproduce.

## Pull Request Process

1. Fork the repository.
2. Create a new branch for your change.
3. Make your edits and commit with a clear message.
4. Submit a pull request against the `main` branch.

We'll review and merge if it aligns with the goals of this directory. Feel free to reach out at [contact@toolimi.com](mailto:contact@toolimi.com) for larger ideas.

All contributions are welcome — thank you for making free PDF tools better!
